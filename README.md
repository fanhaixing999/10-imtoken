你现在是 imToken 官方 10 周年 AI 共创活动参赛选手，需要开发一个高质量 Demo。

**必须严格使用的官方仓库：**
1. UI 组件库：https://github.com/consenlabs/token-ui （必须基于这个仓库的 React + Tailwind 风格，尤其是 @repo/ui 组件）
2. 核心引擎：https://github.com/consenlabs/token-core-monorepo （重点使用 tcx-wasm，特别是 tenth-anniversary 分支或 demo 分支中的 tcx-wasm）

**作品名称**：安心宝 - 新手离线轻钱包
**目标用户**：第一次接触加密货币的绝对新手
**主链**：以 ETH 为主

### 核心设计理念（必须严格遵循）
- 极致新手友好：**绝不出现** “助记词、私钥、公钥、Keystore、地址派生”等专业词汇
- 所有文案使用温暖、安心的语言，例如：“安全备份短语”“我的账户”“安心转账”
- 强调“本地离线生成”，创建钱包时必须突出安全感
- 整体风格：干净、简洁、温暖、信任感强（类似 imToken 但更轻量、更友好）

### 主要页面与功能要求

1. **欢迎页 / 首页**
   - 大标题：“欢迎来到安心宝”
   - 副标题：“第一次用加密钱包，也能安心掌控自己的 ETH”
   - 醒目按钮：“创建我的安心钱包（本地离线生成）”
   - 已有钱包？ → “导入已有账户”（模拟）

2. **创建钱包流程（最重要）**
   - 选择方式：密码模式 / Passkey 模式（推荐 Passkey）
   - 点击创建后显示：“正在你的浏览器本地生成钱包，请保持网络关闭以获得最高安全”
   - 使用 @consenlabs/tcx-wasm 真实调用本地生成（可模拟进度）
   - 生成完成后，进入“安全备份短语”页面（显示 12 或 24 个词）
   - 强制用户手抄确认（至少勾选 3 项确认项）
   - 强烈提示：“建议手抄或打印，永远不要截图或保存在手机/云端”

3. **主资产页**
   - 顶部显示“我的账户” + 简短地址（可点击复制）
   - 大卡片显示总资产（模拟数据：约 0.85 ETH）
   - ETH 为主资产卡片（余额、价值）
   - 底部 Tab：资产 | 收款 | 转账 | 设置

4. **收款页**
   - 大二维码
   - “我的 ETH 收款地址”
   - 一键复制按钮

5. **转账页（安全重点）**
   - 输入接收地址、金额、备注
   - “添加为信任地址” 白名单功能（可保存常用地址）
   - 实时风险提示示例：
     - “此地址曾被标记为高风险，请谨慎”
     - “此合约交互可能有安全风险”
   - 确认转账前要求生物识别（Passkey / 指纹模拟）
   - 显示 Gas 费（简化显示）

6. **设置页**
   - 生物识别开关
   - 查看/重新备份安全短语
   - 信任地址管理
   - “断网模式提醒”

### 技术实现要求
- 使用 Vite + React + TypeScript + Tailwind（基于 token-ui 项目结构）
- 必须集成 `@consenlabs/tcx-wasm` 进行本地钱包创建和签名
- 所有敏感操作（生成、签名）必须在本地完成
- UI 组件尽量使用 token-ui 提供的现成组件，保持官方风格
- 移动端响应式良好
